# Retail Sales EDA
Exploratory Data Analysis of online retail transactions to uncover sales trends, customer behaviour, and product performance using Python.

## Project Overview
This project explores online retail transaction data to identify key patterns in revenue, transaction behaviour, and product performance. The analysis focuses on transforming raw transactional data into meaningful business insights through data cleaning, exploratory analysis, and visualization.

## Objectives
- Analyze overall sales and revenue performance
- Identify transaction patterns and customer purchasing behaviour
- Explore revenue trends over time
- Evaluate product contribution to total revenue
- Segment transactions based on transaction value

## Tools
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Dataset
The dataset contains online retail transaction records, including invoice information, product details, quantity, unit price, customer ID, transaction date, and country.

## Analysis Workflow
1. Data Understanding
2. Data Cleaning
3. Feature Engineering
4. Exploratory Data Analysis
5. Transaction Segmentation
6. Business Insights

## Key Insights

Key findings and business insights will be summarized based on the exploratory analysis results.
